## Features

- Scans ports 20-85 on target hosts
- Displays open ports
- Shows scan duration
- Simple interactive interface

## Usage
```bash
python port_scanner.py
```

Then enter the target when prompted.

## Example Output
```
Enter The Target (domain/IP address): 192.168.1.1
Start scanning 192.168.1.1...
[+] Found Open port: 22
[+] Found Open port: 80

Scan completed in 12.34s
```
